## Run Locally

Clone the project

```bash
  git clone https://github.com/chandansahoo69/Reserve_Now
```

Go to the project directory

```bash
  cd Reserve_Now
```

Install dependencies

```bash
  yarn add
```

Start the server

```bash
  yarn start
```
